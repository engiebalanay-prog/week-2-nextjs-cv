# Week 2 - Next.js CV Website

Personal CV and portfolio website built with Next.js, React, TypeScript, and Tailwind CSS as part of the AI Agent Developer Workshop.

## About

This project is a single-page CV website showcasing my skills, experience, and education as an AI Agent Developer intern. It was built using AI-assisted development workflows with GitHub Copilot.

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- Deployed on Vercel

## Project Structure

```
src/
  app/
    layout.tsx        - Root layout with metadata and fonts
    page.tsx          - Main page composing all CV sections
    globals.css       - Global Tailwind styles
  components/
    Header.tsx        - Navigation header
    Hero.tsx          - Hero section with introduction
    About.tsx         - About me with quick facts
    Skills.tsx        - Skills grouped by category
    Experience.tsx    - Work experience timeline
    Education.tsx     - Education details
    Contact.tsx       - Contact information
    Footer.tsx        - Page footer
```

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
